## Examinator

`Examinator is an online examination platform that offers functionalities for users to create exams, join exams, submit answers, and calculate scores. The platform is completely automated teacher creates exam, and waits for his/her students results. Also exams have sessions which means no answers are lost incase of techinical problem that student might have.`
